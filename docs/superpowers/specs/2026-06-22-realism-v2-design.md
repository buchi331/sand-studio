# 落ち砂サンドボックス — リアル化 v2 設計 (2026-06-22)

## 背景

リアル化 v1（`2026-06-22-realism-design.md`）のフェーズ①〜⑤は実装済み:
粘性差・`life`公開・レンダラファクトリ・Canvas2D粒ゆらぎ・WebGL2 HDRブルーム・
さらに写真水槽合成＋擬似奥行き（上方光＋AO）。本設計はその先の「総合リアル化」。

## ゴール

決定論シム（22テスト）を**一切触らず**、描画側中心で見た目の説得力を底上げする。
ユーザー要望は4軸すべて（物理の見え方・質感・光/空気感・水槽の実在感）。

### スコープ内
- 描画専用の質感/光の強化（濡れ砂・縁の滑らか化・コースティクス・接地影・グレード）。
- 火の上に手続き的な煙/もやを描く（**シム非依存・粒子状態なし**）。
- 背景写真をシェーダに取り込み、水・ガラス越しに屈折させる（実在感の本命）。

### スコープ外
- シムの物理変更・新物質・決定論を壊す変更。
- グリッド解像度の引き上げ（180×320 維持。粗さは描画の滑らか化で代替）。

## 進行（独立した小コミット、各段で typecheck/test/build 緑）

| # | 項目 | 触る場所 | リスク |
| --- | --- | --- | --- |
| 1 | 濡れ砂 | WebGL2 / Canvas2D | 低 |
| 2 | 縁の滑らか化（固体の境界AA） | WebGL2 scene | 低〜中 |
| 3 | コースティクス＋接地影 | WebGL2 composite | 低 |
| 4 | カラーグレード＋ビネット | WebGL2 composite | 低 |
| 5 | 火上の手続き的煙/もや | WebGL2 composite | 低 |
| 6 | 背景写真の屈折 | App/styles + WebGL2 | 中〜大 |

### 1. 濡れ砂
水に接する／水中の砂・石を暗く濃く（濡れ色）。砂が水を吸った質感。
- WebGL2: faux-volume 分岐で隣接/直下の水を検出し `col *= 濡れ係数`、彩度+。
- Canvas2D: 同等の簡易版（隣接水で減光＋青寄せ）。

### 2. 縁の滑らか化
固体（砂/石/壁/植物）の境界をサブセル被覆でAA。nearestのドット角を取りつつ
内部の粒ゆらぎ（VARIATION）は維持。出力アルファに被覆を反映して角を丸める。

### 3. コースティクス＋接地影
- 水中の固体面に、時間で揺れる明るい網目（既存の死蔵コードのcaustic式を復活/簡素化）。
- 山が床・ガラスに落とす柔らかい接地影（AOの局所強化）。

### 4. カラーグレード＋ビネット
合成の最後で写真の色温度に寄せる（軽い暖色リフト＋持ち上げ）＋周辺減光。
全体の馴染みと「奥行き」を出す仕上げ。

### 5. 火上の手続き的煙/もや
火セルの上方に、ノイズベースの上昇するもやを加算的に薄く描く。
粒子状態を持たない純シェーダ表現（`uTime`駆動）。発光ブルームと両立。

### 6. 背景写真の屈折（本命）
現状: 写真は CSS 背景、キャンバスは透明合成で「裏が透ける」だけ。
変更: 写真を WebGL テクスチャとして読み込み、合成シェーダで
**水・ガラス縁の法線に応じて背景UVをずらしてサンプル**し、屈折/歪みを表現。
- キャンバスは水槽内部の部分矩形（`.tank-shell`: left23% top26% w52% h53% of `.room`）に対応。
  背景テクスチャの該当UV範囲を uniform で渡す（レイアウト定数を共有）。
- 空セルは「写真のその位置」を等倍で出す＝従来の透過と同じ見え方を維持。
- 水セルは屈折オフセット＋色被せ。録画は `preserveDrawingBuffer` 維持で担保。
- 非対応/フォールバック時は従来の透過合成のまま（写真未読込なら no-op）。

## 受け入れ
- `npm run typecheck` / `npm test`（22 緑）/ `npm run build` すべて緑、決定論維持。
- WebGL2 経路で録画→再生してフレームが映る（実機 iOS）。
- 見た目: 砂が濡れ、縁が滑らかになり、水越しに背景が歪む。
